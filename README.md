# JLU-JKDK

[JLU自动健康打卡机器人](http://gjz0117.xyz/jkdk)

一次注册即可实现每天早晨自动健康打卡

**注册成功后会有邮件提示**

**每日上午6:00 - 9:00打卡，成功后自动邮件通知**

**若9:00仍未打卡成功，统一发送邮件通知自行打卡**

**不保证100%打卡成功，极小概率会服务器宕机(或打卡系统更新导致程序无法正常运行)**

***

源码中数据库与服务器相关配置参数已隐去，部署时自行填充

服务器配置

centos7 + mysql5.7 + python3.7.5 + flask + uwsgi + nginx
